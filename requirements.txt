# Requirements

## 1. 프로젝트 개요

본 프로젝트는 사용자가 입력한 목표 시간(퇴근 시간, 시험 날짜 등)까지 남은 시간을 계산하고 실시간으로 시각화하는 웹 애플리케이션이다.

사용자가 설정한 시간까지 남은 시간을 초 단위로 계산하여 카운트다운 형태로 보여주며, 해당 시간을 다양한 재미있는 값으로 변환하여 사용자에게 제공한다.

## 2. 주요 기능

### 기능 1 – 목표 날짜/시간 설정

사용자가 퇴근 시간이나 특정 목표 날짜를 직접 입력할 수 있다.

### 기능 2 – 실시간 카운트다운

현재 시간과 목표 시간의 차이를 계산하여 남은 시간을 실시간으로 표시한다.

### 기능 3 – 재미있는 변환 기능

남은 시간을 기반으로 다음과 같은 값으로 변환하여 보여준다.

* 마실 수 있는 커피 수
* 먹을 수 있는 치킨 수
* 예상 가능한 금액

## 3. 페이지 구성

1. 메인 페이지

   * 목표 날짜 및 시간 입력

2. 카운트다운 페이지

   * 목표 시간까지 남은 시간을 실시간으로 표시

3. 변환 결과 페이지

   * 남은 시간을 재미있는 수치로 변환하여 표시

## 4. 사용 기술

* Python
* Flask
* HTML / CSS
* JavaScript
