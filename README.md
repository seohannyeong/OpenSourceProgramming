# My Development Journal API

이 프로젝트는 Flask를 사용하여 구축된 간단한 개발 일지 API입니다.

## 사전 요구 사항 (Prerequisites)

- Python 3.x
- Flask

## 설치 방법 (Installation)

1. 필요한 패키지를 설치합니다.

```bash
pip install flask
```

## 실행 방법 (Usage)

다음 명령어로 애플리케이션을 실행합니다.

```bash
python my_profile.py
```

서버가 실행되면 브라우저에서 `http://localhost:5000`으로 접속하여 확인할 수 있습니다.
